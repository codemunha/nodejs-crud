# EXPRESS CRUD

## Features

- Express CRUD
- Mariadb:10.2.16
- Adminer:4.7.8

## Quick start

### Clone following repo:

```sh
$ git clone https://github.com/codemunha/nodejs-crud.git [PROJECT_NAME]
cd [PROJECT_NAME]
```

## Run Docker Compose

### build & run images 

```sh
$ docker-compose -f docker-compose.yml build --force-rm
$ docker-compose -f docker-compose.yml up
```

### Install and Start project

```sh
$ yarn install
$ yarn start
```
